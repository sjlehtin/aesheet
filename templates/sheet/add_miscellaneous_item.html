When adding a new effect for an item, the special quality needs to be created first, then the effect which will be attached to the special quality.

Example: Adding a ring of protection, you would first create the special quality "Protection L2", then create the effect with the intended armor which would be added to existing special quality "Protection L2".

This will probably be stream-lined a bit later.
